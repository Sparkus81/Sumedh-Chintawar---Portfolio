Mario Kart Blue Shell - Surface Modeling

Personal Project | SolidWorks | 2024
SolidWorks Surface Modeling Organic Shapes Assembly Design Rendering

Project Overview

Recreation of the iconic Blue Shell from Mario Kart using advanced surface modeling techniques in SolidWorks. This project challenged my ability to create complex organic shapes while maintaining clean surface continuity and proper design intent for 3D printing manufacturability.

Technical Approach

  • Surface Modeling: Utilized lofted surfaces, boundary surfaces, and swept features to create smooth organic curves matching the iconic Blue Shell profile
  • Feature Management: Maintained organized feature tree with design intent for easy modifications and parametric updates
  • Assembly Design: Created multi-body parts and sub-assemblies for the shell body, wings, and detail features with proper mate constraints
  • DFM Considerations: Designed with 3D printing in mind, including proper draft angles, wall thicknesses (minimum 2mm), and support-minimizing orientation
  • Detail Features: Incorporated characteristic spikes, panel lines, and geometric details while maintaining clean topology
Software Used: SolidWorks 2024
Primary Techniques: Surface modeling, lofts, boundary surfaces, shell features
Part Count: 4 unique components (shell body, base, 2x wings)
File Format: SLDPRT, STL export for 3D printing
Intended Manufacturing: FDM 3D printing (PLA/ABS)

Design Challenges & Solutions

Challenge: Creating smooth transitions between the spherical shell body and angular spike features while maintaining surface continuity.

Solution: Used curvature-continuous (C2) surface blending with careful control of tangency constraints. Applied "Delete Face" and "Knit" operations to merge surfaces seamlessly.

Challenge: Designing wings that appear aerodynamic while being structurally sound for 3D printing.

Solution: Incorporated internal ribbing (not visible externally) and optimized wall thickness gradients. Used swept boss/base features along 3D guide curves to maintain proper wing profile.

Key Takeaways

  • Strengthened proficiency in advanced surface modeling techniques for organic shapes
  • Learned to balance aesthetic design goals with manufacturing constraints
  • Improved understanding of curvature analysis and surface quality evaluation
  • Developed efficient workflow for creating complex multi-body assemblies
